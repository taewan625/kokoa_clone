git-hub는 statistic webpage를 무료로 제공한다
statistic : HTML, CSS, JS

조건
1. public 저장소여야함
2. branch 이름은 'gh-pages'여야 한다

gh-pages branch를 만들고 나면 이제 publish를 누르고 기다리면 된다

오류 잘나는 실수하는 point
1. 파일명 대문자 소문자 섞인 경우 -> git은 대소문자 구분 못해서 소문자로만 쓰기
2. index.html 파일이 제일 기본으로 있어야 하고 이 파일이 최상위 root에 존재해야함
   만약에 index.html 파일이 readme.md 보다 하위에 존재하면 github site 상에서 readme를 보여줌
   -> 이때 readme 삭제하고 html file 모두 최상위로 올리면 잘 작동함
